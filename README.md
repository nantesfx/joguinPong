# joguinPong

Esse foi um projeto desenvolvido para apresentação em sala de aula na matéria de ALGORITMOS E LINGUAGEM DE PROGRAMAÇÃO. É baseado em um clássico dos anos 80, um jogo de ping pong, básico e funcional. 

Desenvolvido em JS, CSS e HTML.

Os itens abaixos são formados por:

ball.png -> Imagem de uma bola neon que será a bola do usuário. 

paddle.png -> Imagem da barra do usuário. 

paddleIA.png -> Imagem da barra do adversário.

script.js -> Código em JavaScript desenvolvido para que rode o algoritmo/programa.
